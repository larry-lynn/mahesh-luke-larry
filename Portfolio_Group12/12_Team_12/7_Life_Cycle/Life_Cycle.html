Life Cycle

  • The SDLC model that we have used in this project is Iterative and incremental development.
  • In this model, the developers go iterative runs of software development cycle. We started developing this project at the start of the semester, and developed each part of the compiler as and when we studied about it in the class.
  • As new requirements were assessed, the already developed modules sometimes needed some minor changes.
  • As the result of all the changes and modifications, the final product was developed.
